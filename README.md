# tsnode

run ts files in node with type-checking

https://github.com/alexsch01/tsnode

<br>

### How to install

- Node.js 22 (LTS) - v22.18.0 or later

  **OR**

- Node.js v24.3.0 or later
```
npm i -g @alexsch01/tsnode
```

### How to use

```
$ mkdir proj
$ cd proj
$ tsnode --init

[Make a main.ts file]
$ tsnode main.ts
```

### How to update TypeScript

```
$ tsnode --update
```
